= Changelog

== Version 1.11.0 (2026-02-17)

=== Features

* *PR-based Stack ID for AWS Operations*
** Split `repository_name` into separate `owner` and `repo` variables
** Generate PR identifier in format: `{repo}_{pr_number}-merge`
** Use PR identifier as stack ID for AWS status and delete operations
** Ensures consistent stack identification based on PR number rather than branch name
** Add `pr_identifier` as job output for downstream consumption

=== Fixes

* Fix bash parameter expansion syntax in repository name splitting (cannot use `%%` inside `${{ }}`)
* Fix workflow_dispatch input type conversion issues for number and boolean types
* Simplify test workflow expressions to use `github.event.inputs` with `fromJSON()`
* Add realistic default parameters to test_pr_merged.yml

=== Documentation

* Update CLAUDE.md with PR-based stack ID documentation
* Document new variables and outputs
* Add examples for test workflows
